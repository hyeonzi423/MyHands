## 🖥️ 프로젝트 소개

마이핸즈는 두핸즈 구성원들의 성과를 효과적으로 가시화하여 동기부여 및 업무 만족도 향상을 위한 서비스입니다.

## 🕰️ 개발 기간

2025년 1월 5일(일) ~ 1월 17일(금)
